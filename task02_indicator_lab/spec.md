## 技术指标计算方法

### 1. RSI（相对强弱指标）

**原理**：衡量股价变动速度和幅度，判断超买/超卖状态

**计算公式**：
```
RSI = 100 - (100 / (1 + RS))
RS = 平均上涨幅度 / 平均下跌幅度

常用参数：N = 14
- 计算每日涨跌幅
- 分别计算N日内上涨和下跌的平均值
- 计算RS和RSI
```

**判断标准**：
- RSI > 70：超买区域，可能回调
- RSI < 30：超卖区域，可能反弹
- 50为中轴线

---

### 2. MACD（移动平均收敛/发散指标）

**原理**：通过快慢均线的离散、聚合表征当前买卖能量

**计算公式**：
```
MACD线 = EMA(close, 12) - EMA(close, 26)
信号线 = EMA(MACD线, 9)
柱状图 = MACD线 - 信号线

其中：
EMA(n) = 今日收盘价 × (2/(n+1)) + 昨日EMA × (1 - 2/(n+1))
```

**判断标准**：
- MACD > 0：多头市场
- MACD < 0：空头市场
- MACD上穿信号线：买入信号
- MACD下穿信号线：卖出信号

---

### 3. 布林带（Bollinger Bands）

**原理**：通过标准差衡量价格波动范围

**计算公式**：
```
中轨 = SMA(close, N)  # 简单移动平均
上轨 = 中轨 + K × STD(close, N)
下轨 = 中轨 - K × STD(close, N)

常用参数：N = 20, K = 2
```

**判断标准**：
- 价格触及上轨：可能回调
- 价格触及下轨：可能反弹
- 布林带收窄：即将选择方向
- 布林带扩张：趋势强化

---

### 4. ATR（平均真实波幅）

**原理**：衡量价格波动的真实幅度

**计算公式**：
```
真实波幅(TR) = max(
    最高价 - 最低价,
    |最高价 - 昨收|,
    |最低价 - 昨收|
)

ATR = TR的N日指数移动平均
常用参数：N = 14
```

**应用场景**：
- 衡量波动性
- 设置止损位
- 仓位管理

## 验证标准

为确保计算准确性，需与主流金融数据平台对比：
1. **同花顺/东方财富**：对比RSI、MACD值
2. **雪球/腾讯自选股**：对比布林带数值
3. **TradingView**：对比ATR计算

验证方法：
- 选取特定日期，对比指标数值
- 误差允许范围：±0.01（RSI/MACD）/ ±0.01元（布林带/ATR）
